## SSH 文件是什么

- `id_ed25519`
  - SSH 私钥。
  - 不能泄露、不能发给别人、不能提交 Git。
  - 用于免密登录你授权过的服务器或 Git 服务。

- `id_ed25519.pub`
  - SSH 公钥。
  - 可以放到 GitHub、服务器后台或云平台 SSH key 配置里。

- `known_hosts`
  - 曾经连接过的远程主机指纹记录。
  - 不是密码，但保留后能减少重装后 SSH 反复确认主机。

